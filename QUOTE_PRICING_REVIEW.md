# Documento de Precios Provisionales (Cotizador)

Los siguientes precios base han sido configurados en el sistema como valores provisionales para la evolución comercial de `karlosmontoya.dwonky.com`.
Estos valores pueden ser ajustados editando el archivo `src/data/quote-config.ts`.

| Identificador | Proyecto | Precio Base (MXN) |
| :--- | :--- | :--- |
| `landing` | Landing Page | $4,500 |
| `corporate` | Sitio Web Corporativo | $8,000 |
| `ecommerce` | E-commerce | $12,000 |
| `custom` | Sistema a Medida | $15,000 |
| `uxui` | Solo Diseño UX/UI | $2,500 |

| Identificador | Tamaño | Multiplicador |
